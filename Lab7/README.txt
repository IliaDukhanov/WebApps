Запуск
1. Запустить БД командой "mongod.exe --config config.txt" из дериктории "mongodb\bin"
2. Установить компоненты, от которых зависит сервер, командой "npm install" из дериктории "server"
3. Запустить сервер командой "npm start"
4. Установить компоненты, от которых зависит приложение, командой "npm install" из дериктории "client"
3. Запустить приложение командой "npm start"

Поля помеченные знаком "*" не считаются обязательными.

Поле Animal Name может принимать значения: beaver, bison, hippo
Поле Colour может принимать значения: white, purple, black
Поля Width и Height принимают целочисленные значения
